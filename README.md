# 🚀 Proyek Web Pertama GitHub Pages

Ini adalah proyek website statis sederhana yang dibuat untuk mempelajari struktur file repositori dasar dan otomatisasi deployment menggunakan **GitHub Pages**.

## 🛠️ Teknologi yang Digunakan
* **HTML5** - Struktur halaman web.
* **CSS3** - Desain tampilan (*Dark mode style*).
* **JavaScript** - Logika manipulasi tombol interaktif.

## 📦 Isi Repositori
* `index.html` - Halaman utama.
* `style.css` - Desain layout dan animasi.
* `script.js` - Logika interaksi tombol.
* `.gitignore` - Pengaman file lokal agar tidak ikut ter-upload.
